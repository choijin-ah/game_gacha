# Starfall Academy UI 에셋 발주 체크리스트

## 1. 파일 규칙

- 파일명: `{screen}_{purpose}_{variant}_v{number}.png`
- 예시: `weeklyboss_keyvisual_normal_v1.png`, `ui_button_primary_pressed_v1.png`
- 원본 작업 파일은 별도 보관하고 Unity에는 병합된 PNG와 필요한 분할 Sprite만 넣는다.
- UI Sprite는 투명 여백을 최소화하고, 9-slice 대상은 Border 값을 함께 기록한다.
- 아이콘은 같은 세트 안에서 캔버스 크기, 선 굵기, 광원 방향을 통일한다.

## 2. 권장 제작 사양

| 종류 | 권장 크기 | Unity 설정 |
| --- | ---: | --- |
| 가로 키비주얼·배경 | 1920 x 1080 이상 | Sprite, Full Rect, 압축 품질 확인 |
| 캐릭터 전신 | 높이 2048 이상 | 투명 PNG, Tight Mesh |
| 캐릭터 초상 | 512 x 512 | 정사각형, 동일 얼굴 스케일 |
| 배너 | 1600 x 600 이상 | 텍스트 Safe Margin 8% |
| 카드 프레임 | 512 x 768 | 등급별 동일 Border |
| 시스템·콘텐츠 아이콘 | 256 x 256 | 중심 정렬, 투명 PNG |
| 상태 아이콘 | 128 x 128 | 작은 크기 가독성 우선 |
| 파티클·광원 | 512~1024 | Additive 사용 여부 명시 |

## 3. 현재 연결된 핵심 에셋

- 로비 키비주얼과 대체 키비주얼
- 공통 버튼 상태 시트와 로비 아이콘 시트
- 가챠 포털 배경
- 기본 스킬 아이콘 시트
- 기존 캐릭터 전신·초상 원본

연결 경로와 실제 Import 상태는 `Starfall > UI > Style Guide > Assets`에서 확인한다.

## 4. 우선 발주 항목

### P0 — 공통 완성도

- Primary/Secondary/Warning/Danger 버튼 4상태
- 기본·강조·모달·툴팁 패널 9-slice 세트
- 캐릭터·스테이지·보상·장비 카드 프레임
- 속성 6종, 역할 5종, 재화, 시스템 아이콘
- HP·에너지·경험치·점수 게이지
- NEW/PICK UP/LOCK/CLEAR/FREE/RECOMMENDED 라벨

### P1 — 보상 경험

- 3/4/5성 가챠 배경·카드 프레임·광원
- 마법진, 파편, 스파클, 빛줄기
- 승리·패배 타이틀, 별 획득, 레벨업 연출
- 보상 카드 등장과 신규 해금 오버레이

### P2 — 확장 콘텐츠

- 주간 보스 키비주얼·전용 전투 배경·랭크 배지
- 도전의 탑 배경·층 노드·특수 규칙·누적 보상 상자
- 출석판 배경·날짜 프레임·특별 보상 강조
- 우편 프레임·읽음/안읽음·첨부·만료 아이콘
- 장비 세트 및 각성 단계 문양

## 5. 캐릭터 1명당 납품 묶음

- 로비·상세·가챠에 공용 가능한 전신 1장
- 정사각형 초상 1장
- 스토리 스탠딩 1장
- 표정 3~5종은 선택 사항
- 궁극기 컷인 1장은 선택 사항
- 캐릭터 ID와 파일명 매핑표

## 6. 납품 검수

- 투명 가장자리의 흰색/검은색 Halo가 없는가
- 같은 용도의 이미지가 중복 저장되지 않았는가
- 텍스트가 이미지에 불필요하게 구워져 있지 않은가
- 9-slice 확대 시 코너와 장식이 찌그러지지 않는가
- 모바일 축소 크기에서 아이콘 의미가 구분되는가
- 희귀도·성공·경고 상태를 색상 외의 형태/문구로도 구분할 수 있는가
- GUID를 보존하기 위해 Unity 내부 이동을 사용했는가
